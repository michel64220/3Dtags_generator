Installation du nuage sur votre site

  1. Utilisez l'outils de personnalisation ci-dessus pour créer votre nuage de tags
  2. Téléchargez le fichier javascript de votre choix sur le site de son auteur
  3. Placez le ou les fichiers téléchargés dans le même répertoire que votre page HTML
  4. Cochez dans la liste ci-dessous celui que vous avez choisi :
    Stand-alone version 2.8, Full source
    Stand-alone version 2.8, Minified
    jQuery plugin version 2.8, Full source
    jQuery plugin version 2.8, Minified
  5. Copiez et collez le code ci-dessous dans le HEAD de votre fichier HTML

    Avertissement : vous avez choisi une version qui nécessite d'avoir jquery. Après avoir copié-collé ce code dans votre page HTML, remplacez le texte "mettez le nom de votre fichier JS jquery ici" par le nom de votre fichier Jquery

  6. Copiez et collez le code ci-dessous là où vous souhaitez faire apparaître le nuage dans votre page HTML
  7. Copiez et collez le code ci-dessous dans le HEAD de votre fichier HTML
